## Description

A NestJs Api for BAckend Sodtware Web Engineer.

## Tools
NestJs, Typescript, MongoDB, RabbitMQ


## Installation Steps


```bash
$ npm install
```

## Running the app

```bash
# copy the app and cd into the direcory. Then run
$ npm install

# start the app
$ npm run start

# start in watch mode
$ npm run start:dev

# start in production mode
$ npm run start:prod
```

## Testing steps

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
